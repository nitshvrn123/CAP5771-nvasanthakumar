DIARY ENTRY: PROBLEM FORMULATION

Project Objective

In professional cricket, performance is commonly summarized using aggregate statistics such as batting average, strike rate, economy rate, or wickets. While these metrics describe overall ability, they do not explicitly capture how performance changes under high-stakes situations.

In the IPL, competitive pressure varies by tournament stage. League matches allow recovery across a long season, while playoffs and finals introduce elimination risk, heightened scrutiny, and increased competitive intensity.

This project investigates whether later tournament stages (playoffs and finals) are associated with higher performance risk and variability for players compared to league matches.

⸻

Operational Definition of Pressure

Competitive pressure is operationalized using match stage:
	•	League
	•	Playoffs
	•	Final

Rather than collapsing all non-league matches into a single group, I treat league, playoffs, and finals as distinct categories. This preserves tournament structure and allows fine-grained comparison across increasing pressure levels.

⸻

Analytical Focus

The primary focus is not predicting match winners. Instead, I quantify how pressure shifts the distribution of individual performance for both batters and bowlers.

⸻

Batting Outcomes
	•	Runs scored per innings
	•	Probability of a low score (runs < 10)
	•	Variance of runs
	•	Stage-specific shifts in average performance
	•	Batting Pressure Impact Index

⸻

Bowling Outcomes
	•	Wickets per match
	•	Economy rate
	•	Variance of economy
	•	Probability of a high-cost spell
	•	Bowling Pressure Impact Index

⸻

Predictive Modeling Extension (Machine Learning)

In addition to descriptive analysis, this project prepares the dataset for supervised machine learning models that predict player performance under similar match conditions.

Two separate predictive tasks will be developed:

Batting Model
Goal:
Predict expected runs scored in a future match given similar circumstances.

Features:
	•	Historical average runs
	•	Run variance
	•	Low-score probability
	•	Match stage
	•	Season
	•	Venue

Target:
	•	Runs scored

⸻

Bowling Model
Goal:
Predict expected economy rate in a future match under similar circumstances.

Features:
	•	Historical economy rate
	•	Wicket rate
	•	Dot-ball rate
	•	Match stage
	•	Season
	•	Venue

Target:
	•	Economy rate

All models will be trained using only historical matches prior to prediction to avoid data leakage.

⸻

Time Horizon

The dataset covers IPL seasons from 2008 to 2024.
All analysis is conducted at the player–match level.

⸻

Summary

This project combines:

	•	Stage-aware performance comparison
