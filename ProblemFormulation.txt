
PROJECT TITLE: Quantifying Performance Risk in Cricket Using Match Pressure and Fan Sentiment

CENTRAL QUESTION:
Does external pressure, proxied through match importance and fan sentiment, help explain variability and uncertainty in a cricket player’s performance beyond historical performance alone?

PROBLEM FORMULATION:
Traditional cricket performance analysis focuses on averages and historical statistics. However, player outcomes often vary significantly in high-pressure situations such as playoffs and finals. This project investigates whether pressure-related signals—derived from match context and fan reactions—are associated with increased performance variability, even when a player’s underlying skill remains constant.

The project does not attempt to diagnose mental health conditions. Instead, it studies whether observable external pressure indicators correlate with increased uncertainty or risk in player-level outcomes.

IMPORTANCE & CONTEXT:
Understanding performance variability under pressure is valuable for team selection, strategy planning, and risk-aware decision-making in professional sports. While experienced fans may subjectively assess pressure handling, this project aims to quantify these effects using data-driven methods.

SUCCESS CRITERIA:
Success is defined by:
- Demonstrating measurable differences in performance variability across match stages (league vs playoffs vs finals).
- Showing that pressure-related features add explanatory value beyond historical performance averages.
- Producing interpretable visualizations and summaries that clearly communicate these effects.

DATA SOURCES:
1. Structured Data:
   - Ball-by-ball IPL match data (2008-2024) from Cricsheet.
   - Match metadata (season, venue, match stage).
2. Unstructured Data:
   - Twitter/X fan-generated content used to construct a pressure index.

ESSENTIAL VARIABLES:
- Player performance: runs scored, strike rate, dismissal.
- Match context: venue, season, match stage (league, playoffs, final).
- Derived features: recent form, performance variance, stage-based pressure indicators.
- External pressure: fan sentiment intensity and volume.

FEASIBILITY:
The project is feasible within a semester timeline using publicly available datasets. Initial milestones focus on structured data and exploratory analysis, with later extensions incorporating unstructured fan sentiment data.
